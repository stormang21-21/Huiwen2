# Voice Language Switch Commands

Use these commands to switch Roboboy's TTS voice language:

## Available Commands

- `/voice english` - Switch to Singapore English (en-SG-LunaNeural)
- `/voice chinese` - Switch to Chinese Mandarin (zh-CN-XiaoxiaoNeural)
- `/voice tamil` - Switch to Tamil (ta-IN-PallaviNeural)
- `/voice taiwanese` - Switch to Taiwanese (zh-TW-HsiaoChenNeural)
- `/voice male` - Switch to male voice (current language)
- `/voice female` - Switch to female voice (current language)
- `/voice list` - Show all available voices
- `/voice default` - Reset to default (Singapore English female)

## How to Use

1. Send the command in Telegram chat
2. I'll switch the voice in openclaw.json
3. Restart the gateway
4. New voice will be active!

**Note:** Gateway restart required after each switch (takes ~5 seconds)
